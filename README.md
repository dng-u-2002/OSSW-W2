# OSSW-W2
**나만의 사칙연산 계산기 만들기**

## 실행 방법 🔌
터미널(혹은 아나콘다)를 열어서 python calculator.py를 실행한다.   
실행 이후 숫자로 된 입력 및 사칙연산 기호를 입력하여 계산 결과를 확인한다.
